
# FixFlow - Управление дефектами на строительных объектах

Веб-приложение для централизованного управления дефектами на строительных объектах, разработанное на React.

## Функциональные возможности

- **Система ролей**: Инженеры, менеджеры и руководители с разными правами доступа
- **Управление дефектами**: Создание, отслеживание и контроль исправления дефектов
- **Управление объектами**: Контроль строительных проектов и их прогресса
- **Аналитика и отчеты**: Детальная статистика с графиками и экспорт отчетов
- **Адаптивный дизайн**: Работает на десктопе и мобильных устройствах

## Установка

1. Убедитесь, что у вас установлен Node.js (LTS) и npm.
2. Установите зависимости:
```bash
npm install
````

3. Запустите приложение:

```bash
npm run dev
```

4. Откройте браузер по адресу [http://localhost:5173/](http://localhost:5173)

## Роли пользователей

### Инженер

* Регистрация дефектов
* Обновление статусов
* Добавление комментариев
* Просмотр назначенных задач

### Менеджер

* Все возможности инженера
* Назначение исполнителей
* Контроль сроков
* Управление объектами
* Создание отчетов по проектам

### Руководитель/Директор

* Все возможности менеджера
* Просмотр аналитики
* Формирование сводных отчетов
* Общий контроль всех процессов

## Технологический стек

### Язык и фреймворк

* **React 18** — современный фреймворк для SPA
* **TypeScript** — типизированный JavaScript

### Сборщик

* **Vite** — быстрый Dev-сервер с поддержкой HMR и оптимизацией сборки

### UI и стили

* **shadcn/ui** — набор готовых UI-компонентов (button, card, dialog, input, dropdown menu и т.д.)
* **Radix UI** — базовая библиотека компонентов для shadcn/ui
* **TailwindCSS** — утилитарная CSS-библиотека

### Дополнительные библиотеки

* **Lucide-react** — иконки
* **Sonner** — toast-уведомления
* **clsx / tailwind-merge** — утилиты для работы с классами
* **Ручная маршрутизация** через состояние (React Router не используется)

### Экспорт PDF

* Утилита `pdfExport.ts` использует **html2canvas + jsPDF** для сохранения интерфейса в PDF

### Работа с ошибками / уведомлениями

* Обёртка над Sonner: `utils/toast.ts`

## Демо данные

Приложение включает демонстрационные данные для всех разделов:

* Статистика по дефектам
* Информация об объектах
* Графики эффективности
* Примеры отчетов
